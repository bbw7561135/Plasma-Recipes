## Setup
To use this code:
* Star the directory "ComputEM" 
* Download it
* Then run the project following the instructions below

Type on terminal these lines:

```
$ cd /Users/HOME/Downloads
$ cd CoputEM
```

Now choose you topic you want to study (ex. TE_z):

```
$ cd TE_z-cart
$ python TEz_cart.py
```

Enjoy!
